Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing developers to manipulate the HTML and CSS of a page using JavaScript.

How do you select an element from the DOM?

    1. document.getElementById("id") - Selects an element by its ID.
    2. document.querySelector("selector") - Selects the first element matching the CSS selector.
    3. document.querySelectorAll("selector") - Selects all elements matching the CSS selector.
    4. document.getElementsByClassName("className") - Selects elements by class name.
    5. document.getElementsByTagName("tagName") - Selects elements by tag name.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique where you add a single event listener to a parent element instead of multiple listeners to its child elements.

How it works: Events "bubble up" from a child element to its parent, so you can capture events from child elements using the parent.

Why it's useful:
    1. Improves performance by reducing the number of event listeners.
    2. Dynamically handles elements added later to the DOM.
How do you manipulate an elements attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript DOM methods:

Attributes:

    1. element.setAttribute("attributeName", "value") - Sets or updates an attribute.
    2. element.getAttribute("attributeName") - Gets the value of an attribute.
    3. element.removeAttribute("attributeName") - Removes an attribute.

Styles:

    1. element.style.property = "value" - Sets a specific style (e.g., element.style.color = "red").
    2. element.classList.add("className") - Adds a CSS class.
    3. element.classList.remove("className") - Removes a CSS class.
    4. element.classList.toggle("className") - Toggles a CSS class on or off.
